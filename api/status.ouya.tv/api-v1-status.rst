===========================================
``GET http://status.ouya.tv/api/v1/status``
===========================================

Check if the internet connection is working by fetching an empty file.

HTTP request
============
Protocol
  ``http``
Method
  ``GET``
Host
  ``status.ouya.tv``
Path
  ``/api/v1/status``
Headers
  None of the standard headers.

  ::

    User-Agent: Ouya-Connection-Test


HTTP response
=============
Status code
  204 No Content
